# Dafuq am I doing ?

A simple CLI script returning your currently assigned issues on Mantis bugtracker

## Setting up

* Clone this rep
* ``` npm install ```
* Edit config/default.json with your credentials. Provide the WSDL url.
* ``` npm link ```

## Use it

``` daid ```
